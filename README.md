# 🎯 翻牌認字遊戲

一個適合 4 歲幼兒的互動式翻牌認字遊戲網頁應用！

## ✨ 功能特點

- 🎴 **10 張互動卡片** - 點擊翻開顯示文字或圖片
- 🎨 **色彩鮮艷的卡通風格** - 吸引幼兒注意力
- 📱 **響應式設計** - 手機、平板、電腦都能玩
- 🔊 **音效提示** - 翻牌時有可愛的音效
- 🎉 **慶祝動畫** - 完成遊戲會有彩帶效果
- ✏️ **自定義卡片** - 可以修改卡片內容
- 💾 **本地儲存** - 設定會自動保存

## 🚀 快速開始

### 方法一：直接打開
1. 雙擊 `index.html` 文件
2. 在瀏覽器中開始遊戲！

### 方法二：使用本地伺服器
```bash
# 使用 Python
python3 -m http.server 8000

# 使用 Node.js (需要 http-server)
npx http-server -p 8000

# 然後訪問 http://localhost:8000
```

## 📝 如何修改內容

### 方式一：使用遊戲內的自定義功能（推薦）
1. 點擊「✏️ 自定義卡片」按鈕
2. 選擇要修改的卡片編號
3. 選擇內容類型（文字或圖片）
4. 輸入內容後點擊「💾 儲存卡片」

### 方式二：修改 config.json
1. 打開 `config.json` 文件
2. 修改卡片內容：

```json
{
  "cards": [
    {
      "type": "text",
      "content": "貓",
      "color": "#FF6B6B"
    },
    {
      "type": "image",
      "content": "cat.jpg",
      "color": "#4ECDC4"
    }
  ],
  "soundEnabled": true,
  "flipDelay": 2000
}
```

**字段說明：**
- `type`: 卡片類型 (`text` 或 `image`)
- `content`: 文字內容或圖片網址/路徑
- `color`: 卡片顏色（十六進制顏色碼）
- `soundEnabled`: 是否開啟音效 (`true`/`false`)
- `flipDelay`: 翻牌後自動翻回的延遲時間（毫秒）

## 🖼️ 使用圖片

### 網絡圖片
```json
{
  "type": "image",
  "content": "https://example.com/cat.jpg"
}
```

### 本地圖片
1. 將圖片放在與 `index.html` 相同的資料夾
2. 在配置中使用圖片檔名：

```json
{
  "type": "image",
  "content": "cat.jpg"
}
```

## 🎮 遊戲說明

1. **開始遊戲**：點擊任意卡片翻開
2. **學習認字**：卡片會顯示文字或圖片，2 秒後自動翻回
3. **重置遊戲**：點擊「🔄 重置遊戲」重新開始
4. **自定義**：點擊「✏️ 自定義卡片」修改內容
5. **音效開關**：點擊右上角的 🔊/🔇 按鈕

## 🛠️ 技術細節

- **純前端**：無需後端，單一 HTML 文件
- **響應式**：使用 CSS Grid 和 Flexbox
- **本地儲存**：使用 localStorage 保存設定
- **動畫效果**：CSS 3D 翻轉動畫
- **音效**：Web Audio API 生成

## 🐛 故障排除

**問題：卡片不翻轉**
- 解決：重新整理頁面（F5）

**問題：音效不播放**
- 解決：點擊右上角 🔊 按鈕開啟音效

**問題：自定義不保存**
- 解決：清除瀏覽器快取後重試

**問題：圖片不顯示**
- 解決：檢查圖片路徑是否正確，或圖片網址是否可訪問
